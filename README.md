# TDS JS Task

Create a currency conversion tool

## Stack

- Vite + React + Typescript
- Tanstack Query
- ShadCN + TailwindCSS

## How to run

- clone the repo
- create a .env file and add the api key

```
CURRENCY_API_KEY=yourCurrencyKey
```

```bash
npm i
npm run dev
```
